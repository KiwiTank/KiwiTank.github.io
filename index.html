Enigma simulator

This is a project I did for my MSc in AI to create a working emulator of the German Enigma
coding machine from WWII. Once the machine is created and set up, it then decodes some encrypted
text, including text encoded using missing or altered plug and rotor settings for which the system
can determine the missing settings and apply them to the decryption

Sudoku solver

The game of sudoku is well known to many. You are given a 9x9 grid with some fixed values, and you must fill the empty cells.
This agent solves a standard sudoku puzzle using a backtracking depth-first search with constraint propagation algorithm.
